# SHEEP_HOME Game Website

This is the GitHub Pages website for the SHEEP_HOME farming game. The website showcases the game's features, provides download links, and gives information about the game.

## Files Structure

- `index.html` - Main HTML file for the website
- `styles.css` - Stylesheet for the website
- `script.js` - JavaScript for interactivity
- `images/` - Directory for images (to be added)

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Sections for game features, power-ups, and download options
- Modern, clean design with farm-themed colors

## How to Deploy

This website is designed to work with GitHub Pages:

1. Push these files to the `main` branch of the `Chenboda01/SHEEP_HOME` repository
2. Go to the repository settings
3. Enable GitHub Pages from the `/ (root)` source
4. The website will be available at https://Chenboda01.github.io/SHEEP_HOME/

## Customization

To customize the website:

1. Edit the content in `index.html` to update text and links
2. Modify `styles.css` to change colors, fonts, or layout
3. Update `script.js` to change interactive behaviors

## Images

To add images to the website:

1. Create an `images` folder in the repository
2. Add your images (logo, screenshots, etc.) to this folder
3. Update the HTML to reference these images
